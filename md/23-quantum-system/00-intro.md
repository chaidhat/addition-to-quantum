# Quantum System

A quantum system is defined by:

1.  A [<span class="underline">Hilbert space</span>](#hilbert-spaces) $H$

2.  A Hamiltonian $\hat{H}$ generating time evolution via the [<span class="underline">Schrödinger equation</span>](#schruxf6dingers-equation-hamiltonian).

The Hilbert space’s dimensions defines the fixes the spin of the system.
