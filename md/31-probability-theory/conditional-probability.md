# Conditional Probability

from a joint distribution → if I select $y$ for $Y$ with
$P(Y)\neq 0$

What is the probability $x$ given $y$

$$\begin{gather*}
    P(X|Y)=\frac{P(X,Y)}{P(Y)}
\end{gather*}$$

Special case

If X, Y are independent (if X occurs, Y has no effect, and vice versa)

$$\begin{gather*}
    P(X,Y)=P(X)P(Y)
\end{gather*}$$
