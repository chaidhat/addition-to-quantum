# Linear Independence

Let “linearly independent states” be states that you cannot create
another linearly independent state from multiplying and adding another
linearly independent state.

a cleaner definition is $\left\lvert \psi_1 \right\rangle,\left\lvert \psi_2 \right\rangle,…\left\lvert \psi_n \right\rangle$ are
linearly independent if the only way to get

$$\begin{gather*}
    c_1\left\lvert \psi_1 \right\rangle+c_2\left\lvert \psi_2 \right\rangle+\cdots+c_n\left\lvert \psi_n \right\rangle=0
\end{gather*}$$

is having $c_i=0$ for all ($\forall$) $c_i$.

Example:

Given $c$ is just a regular number,

$$\begin{gather*}
    \left\lvert + \right\rangle=c\left\lvert - \right\rangle\quad\text{will never happen how much you try}
\end{gather*}$$

and vice versa.
