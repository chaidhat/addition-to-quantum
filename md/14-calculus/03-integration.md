# Integration

The opposite of [<span class="underline">differentiation</span>](#differentiation) is integration. This is the Riemann sum.

$$\begin{gather*}
    \int_\alpha^\beta f(t)\cdot dt=\lim_{n\to\infty}\sum_{i=1}^n f(t_i)\cdot (t_i-t_{i-1})
\end{gather*}$$

Integration is much more hellish. You don’t need to know integration for
this.
