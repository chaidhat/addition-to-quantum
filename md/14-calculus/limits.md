# Limit

Let a "limit" be a function that slowly moves something towards a value. For example, if $h$ approaches $\infty$, the equation below evaluates to a value. It’s helpful when if we just simply set $h=0$ or $h=\infty$, then we divide by zero or for some reason can’t compute $h$.

$$\lim_{h\to \infty} 3+\frac{5}{h}=3$$
