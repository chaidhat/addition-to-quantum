# Hermitian Operator

An [[operators|operator]] is [[hermitian-matrix|Hermitian]] if and only if the [[adjoint]] of the operator is
equal to itself.

Properties

- all [[eigenvectorseigenvalues|eigenvalues]] of a Hermitian operator are real

- All [[eigenvectorseigenvalues|eigenvectors]] of a Hermitian operator form an [[orthonormality|orthonormal]] [[basis-states|basis]]

- [[observables]] are usually some kind of Hermitian operator

The [[eigenvectorseigenvalues|eigenvalues]] represent possible measurement outcomes. [[orthonormality|Orthonormal]]
[[eigenstates]] $\left\lvert \alpha \right\rangle$ mean that these are definite value states
