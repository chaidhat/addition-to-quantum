# Diagonalization

A matrix is diagonalizable if it has [<span class="underline">linearly independent</span>](#linear-independence) [<span class="underline">eigenvectors</span>](#eigenvectorseigenvalues)
which occurs if the geometric multiplicity (dimension of the [<span class="underline">eigenspace</span>](#eigenspaces))
equals the algebraic multiplicity (root multiplicity in the
[<span class="underline">characterstic polynomial</span>](#characteristic-polynomial)) for every [<span class="underline">eigenvalue</span>](#eigenvectorseigenvalues).

If matrix $A$ is diagonalizable then it can be written as

$$\begin{gather*}
    A=PDP^{-1}
\end{gather*}$$

Where $D=diag(\lambda_1, … , \lambda_n)$
