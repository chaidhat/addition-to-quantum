# Eigenbasis

An eigenbasis of [<span class="underline">operator</span>](#operators) $\hat{A}$ is a set of [<span class="underline">eigenvectors</span>](#eigenvectorseigenvalues)/[<span class="underline">eigenstates</span>](#eigenstates) that form a [<span class="underline">basis</span>](#basis-states) for the [<span class="underline">Hilbert space</span>](#hilbert-spaces).

Every [<span class="underline">Hermitian operator</span>](#hermitian-operators) has an [<span class="underline">eigenbasis</span>](#eigenbasis).
