# Eigenvector/Eigenvalue

<https://www.youtube.com/watch?v=PFDu9oVAE-g>

$$\begin{gather*}
    A\vec{v}=\lambda\vec{v}
\end{gather*}$$

Basically if I apply matrix transformation $A$ to a vector $\left\lvert v \right\rangle$
and it still points in the same direction, it is an Eigenvector. It
could be multiplied by a scalar value $\lambda$ (Eigenvalue) after
applying the transform, which means its [<span class="underline">magnitude</span>](#magnitude) need not be preserved
by the transform.

The set of all vector $\left\lvert v \right\rangle$’s are Eigenvectors of transformation
$A$.

An [<span class="underline">Eigenstate</span>](#eigenstates) are states instead of [<span class="underline">vectors</span>](#vectors), [<span class="underline">operators</span>](#operators) instead of
matrices but are the same thing.
