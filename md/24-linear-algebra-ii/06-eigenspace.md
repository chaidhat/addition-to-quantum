# Eigenspace

An eigenspace of operator $\hat{A}$ is the set of all [<span class="underline">eigenvectors</span>](#eigenvectorseigenvalues)/[<span class="underline">eigenstates</span>](#eigenstates) with the same [<span class="underline">eigenvalue</span>](#eigenvectorseigenvalues) plus the zero vector.

Example

If [<span class="underline">operator</span>](#operators) $\hat{A}$ has [<span class="underline">eigenvalue</span>](#eigenvectorseigenvalues) $\lambda$, then the $\lambda$-eigenspace of the operator $\hat{A}$ is

$$\{\left\lvert \psi \right\rangle:\hat{A}\left\lvert \psi \right\rangle=\lambda\left\lvert \psi \right\rangle\}$$
