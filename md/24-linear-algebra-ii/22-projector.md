# Projector

Projectors are
[<span class="underline">operators</span>](#operators)
with a special property

$$\begin{gather*}
    \hat{\Pi}^2=\hat{\Pi}
\end{gather*}$$

A projector operator (aka. a projector) is a
[<span class="underline">projection</span>](#projections)
that

$$\begin{gather*}
    \hat{\Pi}_u=\frac{\left\lvert u \right\rangle\left\langle u \right\rvert}{\left\langle u|u \right\rangle}
\end{gather*}$$

Usually, by convention, we pick the representative state $\left\lvert u \right\rangle$ to be normalized, i.e., $\left\langle u|u \right\rangle=1$.
So sometimes it can be written as
$$\hat{\Pi}_u=\left\lvert u \right\rangle\left\langle u \right\rvert\quad \text{if} \quad\left\langle u|u \right\rangle=1$$

once i apply it to a state $v$, i get the projection of that state
$v$ on state $u$
