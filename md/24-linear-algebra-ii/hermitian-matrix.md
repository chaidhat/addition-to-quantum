# Hermitian Matrix

Let a “Hermitian matrix” be a matrix that equals its own [[conjugate]]
transpose

If and only if $A$ is a matrix and is Hermitian, then this is true

$$\begin{gather*}
    A=A^+
\end{gather*}$$

This means a bra and ket are Hermitian matrices of each other!

Another example:

$$\begin{gather*}
    \left\lvert \psi \right\rangle=\begin{bmatrix}a\\b\end{bmatrix}\quad \left\lvert \phi \right\rangle=\begin{bmatrix}c\\d\end{bmatrix}\Rightarrow\left\langle \phi \right\rvert=\begin{bmatrix}c^* & d^*\end{bmatrix}
\end{gather*}$$
