# Hermitian Operator

An [<span class="underline">operator</span>](#operators) is [<span class="underline">Hermitian</span>](#hermitian-matrix) if and only if the [<span class="underline">adjoint</span>](#adjoint) of the operator is
equal to itself.

Properties

- all [<span class="underline">eigenvalues</span>](#eigenvectorseigenvalues) of a Hermitian operator are real

- All [<span class="underline">eigenvectors</span>](#eigenvectorseigenvalues) of a Hermitian operator form an [<span class="underline">orthonormal</span>](#orthonormality) [<span class="underline">basis</span>](#basis-states)

- [<span class="underline">observables</span>](#observables) are usually some kind of Hermitian operator

The [<span class="underline">eigenvalues</span>](#eigenvectorseigenvalues) represent possible measurement outcomes. [<span class="underline">Orthonormal</span>](#orthonormality)
[<span class="underline">eigenstates</span>](#eigenstates) $\left\lvert \alpha \right\rangle$ mean that these are definite value states
